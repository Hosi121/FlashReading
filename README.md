# FlashReading

## 概要

技育キャンプ2024＠vol13，チーム「よ～ねんき」の発表作品です．

## 説明

長文を受動的に読めるように工夫したアプリケーション．単語を入力することで，生成系ＡＩが自動的に長文を作成する．そのため，任意の言語で学習が可能．
